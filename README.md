# AI Resume Screening System

This project is an AI-based Resume Screening System that analyzes resumes and generates ATS-friendly outputs.

## 🚀 Features
- Resume parsing using Python
- ATS-friendly formatting
- Candidate profile generation
- GUI built with PyQt6

## 🛠️ Technologies Used
- Python
- PyQt6
- PyPDF2
- FPDF

## 📂 Project Structure
- resume.py → Main application file

## ▶️ How to Run
1. Install dependencies:
   pip install PyQt6 PyPDF2 fpdf

2. Run the project:
   python resume.py
